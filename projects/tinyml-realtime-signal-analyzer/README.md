# TinyML Realtime Signal Analyzer (STM32 + TFLM)

## Goal
Run lightweight deep learning inference on STM32 in realtime for biomedical and industrial signal classification.

## Primary Use Cases
- Biomedical: ECG arrhythmia detection, EMG motion class recognition
- Industrial: vibration anomaly detection for predictive maintenance

## Recommended Starter Hardware
- STM32 Discovery Kit with onboard MEMS (accelerometer/mic)

## Milestone Overview
1. M0: Platform bring-up and sensor data capture
2. M1: Data pipeline and baseline model
3. M2: TFLM integration and realtime inference
4. M3: On-target validation and latency/power tuning

See `docs/` for detailed execution plan and verification criteria.
